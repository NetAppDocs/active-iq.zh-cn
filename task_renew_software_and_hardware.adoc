= 通过Digital Advisor确定存储系统的软件和硬件更新方案
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
您可以主动识别存储系统中已过期或将在未来 6 个月内到期的软件和硬件，并发送硬件和软件续订请求。

.步骤
. 单击 * 计划 * 小工具中的 * 续订 * 。
. 选择要续订的存储系统，然后单击“续订”。
. 或者，也可以提供其他注释。
. 单击 * 发送 * 。
