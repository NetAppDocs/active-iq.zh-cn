= 确定是否建议在 Digital Advisor 中为您的存储系统进行系统刷新
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
如果要确定是否建议为您的 Tech Support 合同或硬件进行系统刷新，您可以使用系统刷新选项。

您可以通过生命周期规划（*存储* > *生命周期规划*）或通过 Digital Advisor（*仪表板* > *规划小部件* > *系统刷新*）访问此功能。

有关此功能的详细信息，请参见 Console 文档中的link:https://docs.netapp.com/us-en/bluexp-economic-efficiency/use/tech-refresh.html["评估系统刷新"]。
