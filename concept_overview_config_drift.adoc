[role="lead"]
Config Drift 功能有助于主动识别系统上的配置差异。您可以根据需要将系统或集群配置与使用生成 AutoSupport 数据的系统创建的模板进行比较。

未来版本中将添加更多基于社区智慧的模板和其他选项。

您可以查看下图，了解用于确定配置差异并生成报告的工作流。
